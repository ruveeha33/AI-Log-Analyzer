# AI Log Analyzer

AI Log Analyzer is a Linux based web application built using Python Flask, HTML, CSS, and JavaScript.

The project analyzes Linux system logs and provides simplified AI style explanations for important events such as failed login attempts, disk usage warnings, and system activities.

## Features

- Linux log analysis
- Flask backend API
- Frontend dashboard
- AI based log explanations
- Error and warning classification

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Ubuntu Linux
- VMware
